## Build üöÄ

**1. Clone the repo**
```javascript
gh repo clone saviomartin/markdown-editor
```

**3. Entering project directory**
```javascript
cd markdown-editor/
```

**3. Installing Dependencies**
```javascript
npm install
```

**4. Starting server**
```javascript
npm start
```

**4. Production Build For Deployment**
```javascript
npm run build
```
